### [RAG Alzheimer Assistant](https://github.com/WeinerGero/RAG-alzheimer-assistant)

A local RAG system for exploring scientific literature on therapeutic targets in Alzheimer's disease.

The project retrieves PubMed publications, combines semantic and lexical search, deduplicates evidence at the article level, and generates source-grounded answers through a local LLM.

`Python` `PubMed API` `ChromaDB` `BM25` `Reciprocal Rank Fusion` `Ollama` `Mistral-Nemo` `Streamlit`

[Open repository →](https://github.com/WeinerGero/RAG-alzheimer-assistant)
